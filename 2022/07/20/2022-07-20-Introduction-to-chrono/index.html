Introduction to chrono

发表于2022-07-20|更新于2022-07-20|课程相关机械设计基础
|阅读量:


  本文很多内容参考了官方文档1知乎博客2，后者讲的很好，推荐阅读。chrono库定义了三种主要类型和相关的函数，分别是time points（时间点），clocks（时钟），durations（时间长度），下面先介绍这几个概念：

三个基本模板类

  1. Clock：时钟包括一个starting point，也叫epoch，和一个tick rate。库中有很多预定义的时钟类型，例如std::chrono::system_clock。system_clock是代表系统时间的时钟，从C++20开始它的epoch被确定为Unix Time，即Unix时间戳，而在之前的版本中可能是unspecified的。其他种类的时钟如下表所示：
时钟类型 描述
system_clock(C++11) wall clock time from the system-wide real-time clock (class)
steady_clock(C++11) monotonic clock that will never be adjusted (class)
high_resolution_clock(C++11) the clock with the shortest tick period available (class)
is_clock / is_clock_v(C++20) determines if a type is a Clock (class template) (variable template)
utc_clock(C++20) Clock for Coordinated Universal Time (UTC) (class)
tai_clock(C++20) Clock for International Atomic Time (TAI) (class)
gps_clock(C++20) Clock for GPS time (class)
file_clock(C++20) Clock used for file time (typedef)
local_t(C++20) pseudo-clock representing local time (class)
  1. Time point：time_point是一个模板类，模板参数有两个，分别是一个时钟类型和一个时间长度，时间点的实现方式是从时钟的epoch计算经过该duration的时间长度。该类重载了各种运算操作符，但是没有重载<<和>>，所以需要使用std::put_time转换为string输出。
  1. Duration：也是chrono定义好的模板类，模板参数包括了一个Rep和一个Period，前者表示ticks的数据类型，例如double，int，float等，后者是两个ticks之间的周期长度，有很多预定义的长度，如下图：
